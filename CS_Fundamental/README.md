# Computer Science Fundamentals

Study track for core Computer Science foundations used in software engineering interviews and production systems.

## Goal

Build a practical mental model for:

- Choosing data structures and algorithms under constraints.
- Understanding how OS, CPU, memory, network, and storage affect software behavior.
- Explaining system trade-offs clearly in interviews.
- Debugging production issues with fundamentals instead of guesswork.

## Suggested Order

1. Data structures and algorithms.
2. Operating systems.
3. Networking.
4. Databases.
5. Distributed systems.
6. Computer architecture.
7. Security basics.

For C++ systems roles, prioritize OS, networking, architecture, and performance-sensitive data structures.
