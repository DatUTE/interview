## Highest Priority Topics

For this JD, prioritize:

1. **C/C++ embedded software**: RAII, memory safety, concurrency, object-oriented design, code quality.
2. **Embedded Linux**: boot flow, device tree, drivers, processes, logging, systemd, cross-compilation.
3. **Networking**: TCP/IP, UDP, Bluetooth, DNS, Wi-Fi, reconnect, buffering, offline/online sync.
4. **Camera/device reliability**: storage, battery, thermal, performance, crash recovery.
5. **Testing**: unit, integration, system-level tests, hardware-in-loop, CI/CD.
6. **Security and privacy**: secure boot, encrypted evidence, chain of custody, least privilege.
7. **System design**: camera device to cloud evidence pipeline, trade-offs, failure modes.
